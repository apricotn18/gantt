# WBS & Gantt Chart

React + TypeScript で構築したWBS（作業分解構造）＆ガントチャートアプリ。

## 機能

- **WBSパネル** — タスク名・開始日・終了日・進捗をリスト表示
- **ガントチャート** — タスクをバーで可視化、TODAY線で現在位置を表示
- **親・子タスク構造** — 親タスクの展開／折りたたみ
- **タスク追加・編集・削除** — モーダルフォームおよび右クリックメニューから操作
- **ズーム** — 1日あたりの幅を調整（−／＋ボタン）
- **スクロール同期** — WBSとガントチャートを連動スクロール

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 技術スタック

- React 18
- TypeScript
- Vite 5
